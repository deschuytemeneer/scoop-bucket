My personal bucket for [Scoop](https://scoop.sh).

## Installation

To add this bucket, run:

```ps1
scoop bucket add deschuytemeneer https://github.com/deschuytemeneer/scoop-bucket
```

To install an app, do:

```ps1
scoop install <app-name>
```
